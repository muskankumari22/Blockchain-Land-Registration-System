# Blockchain-Land-Registration-System
A Solidity smart contract for a blockchain-based land registration system developed using Remix IDE.
This smart contract provides a secure and transparent way to register land records, transfer ownership, verify ownership details, and maintain immutable land information on the Ethereum blockchain.

Note: This project is an academic implementation created for learning and demonstration purposes.

Features

Register new land records (Admin only)

Transfer land ownership securely

Retrieve land details

Check land registration status

Delete land records (Admin or Current Owner)
View all lands owned by a user
Event logging for registration, transfer, and deletion

Immutable blockchain-based ownership records

How to Run
Open Remix IDE.

Create a new Solidity file named LandRegistry.sol.

Copy the smart contract code into the file.

Compile using Solidity Compiler version 0.8.x.

Deploy the contract using the Remix VM Environment.

Test the available functions.
